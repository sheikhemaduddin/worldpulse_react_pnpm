# WorldPulse — News Portal (React + pnpm)

Real-time news portal with breaking ticker, category browsing, article reader, search, trending with live view counters.

## Features
- Live breaking news ticker (CSS animation)
- Category filtering (World, Tech, Business, Science, Sports, Health)
- Article reader with related articles
- Trending page with live view counter updates (Chart.js + real-time interval)
- Search
- Bookmark system (Redux)
- Dark mode toggle

## Setup
```bash
pnpm install
pnpm start
pnpm build
```
